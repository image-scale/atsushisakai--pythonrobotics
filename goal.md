# Goal

## Project
pythonrobotics — a Python robotics algorithms collection

## Description
A collection of robotics algorithms implemented in Python, covering path planning, localization, SLAM, path tracking, and mapping. The library provides easy-to-understand implementations of widely-used robotics algorithms with minimal dependencies (numpy, scipy, matplotlib). Each algorithm can be run standalone for visualization and testing.

## Scope
- Core utilities: angle modulo functions, rotation matrices, plotting helpers
- Path Planning algorithms: Dijkstra, A*, RRT, Potential Field, PRM
- Localization: Extended Kalman Filter (EKF)
- SLAM: Iterative Closest Point (ICP) matching
- Path Tracking: Pure Pursuit control
- Mapping: K-means clustering

The implementation focuses on the core algorithms that define the project's main functionality, delivering practical, working implementations suitable for educational purposes and robotics development.
